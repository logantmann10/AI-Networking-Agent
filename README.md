# Phi Chi Theta Alumni Search

Phi Chi Theta Alumni Search is a web app that helps users search the alumni database using natural language or simple keywords.

Users can search by:
- Name
- Employer
- Position / Role
- City
- Pledge Class
- Graduation Year

Examples:
- Deloitte
- Chicago
- consulting alumni
- Alpha pledge class
- class of 2018
- people in finance

## Features

- Search bar frontend
- Node/Express backend
- Reads alumni data from Excel
- Fuzzy matching for close results
- Optional Gemini AI query parsing
- Returns "No result found." when nothing matches
- Purple and gold custom UI with Phi Chi Theta branding
